# Session 3 Forms Inventory

This is the appraiser's authoritative list of forms for Session 3 review and authoring. Approximately 80 forms across 10 functional categories. Includes both forms not yet in canonical library and refinements/cousin-forms to existing entries (e.g. dressing table refines the dressing-related cousin-form set with washstand and dresser, sideboard/buffet/server already have entries that this list contributes additional cousin-form context to).

Each form requires `distinguishing_features` and `regional_period_notes` authoring at the calibration depth established for `form_pump_organ_cabinet` and `form_washstand` (see existing canonical entries in `lib/constraints/forms.ts` for the depth target). Some forms will additionally warrant `cousin_form_contrasts`, `dimensional_thresholds`, and/or `common_aliases` per the schema additions made during Session 2 mid-session schema work.

## Date envelope scoping note

Some entries below are tagged with broad date framings ("Antiquity → present", "Medieval → present") that reflect the form's general historical existence rather than its production envelope within Proof Sleuth's American antique furniture domain. During Session 3 authoring, these need to be bounded to the relevant timeframe (typically "c. 1700 American examples → present" or similar). Examples flagged for date envelope review:

- Bedstead, Folding Chair, Workbench, Dining Table — currently "Antiquity → present"
- Trestle Table, Refectory Table, Cupboard — currently "Medieval → present"
- Credenza — currently "Renaissance → present"
- Highboy Dresser, Lowboy Dresser — currently "c. 1700 → 1800" (worth confirming against documented Colonial Revival production into the 1900s)
- Pump Organ — currently "c. 1850 → 1920" in this list vs. "1870–1900" in the existing canonical entry; reconcile during Session 3
- Washstand — currently "c. 1800 → 1920" in this list vs. "1820–1900" in the existing canonical entry; reconcile during Session 3

## Inventory

### Seating Forms (Structurally Distinct)

**Armchair**
- Identifiers: fixed arms integrated into seat frame; wider seat than side chair; front legs carry arm supports
- Date: Antiquity → present

**Side Chair**
- Identifiers: no arms; lighter frame; narrower seat; often part of sets
- Date: Antiquity → present

**Windsor Chair**
- Identifiers: solid seat with drilled stick back; splayed legs; wedged tenons
- Date: c. 1720 → present (declines after 1850, revivals later)

**Ladder-back Chair**
- Identifiers: horizontal slats mortised into rear uprights; rush or plank seat
- Date: c. 1700 → 1900 (rural persistence beyond)

**Rocking Chair**
- Identifiers: curved rockers attached to legs; otherwise standard chair structure
- Date: c. 1725 → present

**Morris Chair**
- Identifiers: adjustable reclining back via notched arms; loose cushions
- Date: c. 1880 → 1920 (revivals later)

**Bergère Chair**
- Identifiers: fully upholstered enclosed arms and sides; exposed wood frame
- Date: c. 1720 → 1850 (revival later)

**Wingback Chair**
- Identifiers: high back with projecting side "wings"; enclosed sides
- Date: c. 1700 → present

**Club Chair**
- Identifiers: low, deep, fully upholstered; thick arms; heavy proportions
- Date: c. 1890 → present

**Barrel Chair**
- Identifiers: continuous curved back forming semicircle; integrated arms
- Date: c. 1900 → present

**Slipper Chair**
- Identifiers: armless; low seat height; short legs
- Date: c. 1850 → present

**Parsons Chair**
- Identifiers: fully upholstered rectilinear form; no exposed frame
- Date: c. 1930 → present

**Corner Chair**
- Identifiers: triangular seat; back wraps two sides; designed for corner placement
- Date: c. 1750 → 1850

**Desk Chair (traditional)**
- Identifiers: often swivel base; casters; low arms or none
- Date: c. 1800 → present

**Bentwood Chair**
- Identifiers: steam-bent continuous wood members; light frame
- Date: c. 1850 → present

**Folding Chair (wood/metal canonical form)**
- Identifiers: hinged frame; collapsible planar structure
- Date: Antiquity → present

**Campaign Chair**
- Identifiers: knock-down construction; leather sling seat; turned legs
- Date: c. 1800 → 1900

**Settle**
- Identifiers: bench with high back and arms; enclosed or paneled sides
- Date: c. 1600 → 1850

**Settee**
- Identifiers: elongated upholstered seating for 2+; fixed back; exposed or upholstered frame
- Date: c. 1700 → present

**Sofa**
- Identifiers: long upholstered seating; continuous back; fixed frame
- Date: c. 1700 → present

**Chaise Longue**
- Identifiers: extended seat for reclining; one continuous body-support surface
- Date: c. 1750 → present

**Recamier**
- Identifiers: back and foot ends of equal height; no full back
- Date: c. 1800 → 1850 (revivals later)
